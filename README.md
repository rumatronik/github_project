# PROJECT: Bikeshare Data Analysis
The goal of the Bikeshare project is to use Python to explore data related to bike share systems. 

## General
Template of Python file was offered by Udacity and gave a rough guardrails. Then it has been modified by Marc Ruhmann.
Additionally the bikeshare data files were provided by Udacity.

**Date created:**  
*17.11.2025*

## Setup
* Get Python (Version 3.10 or higher)
* Install Libraries Numpy and Pandas
* Download relevant CSV data for analysis
* Run bikeshare.py file with Python

>**NOTE**: *The CSV files have to be stored locally because they are too big. Please download them directly from the Udacity project.*

## List of files included in the project
* bikeshare.py
* chicago.csv
* new_york_city.csv
* washington.csv
